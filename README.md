### 🚀 Featured Projects · Projetos em Destaque

#### [icompras](https://github.com/JerryAdrianoo/icompras)
🇺🇸 Event-driven microservices platform (customers, orders, billing) built with **Spring Boot** and **Apache Kafka**. Services communicate asynchronously through Kafka topics; invoices are generated with JasperReports and stored in **MinIO** object storage.
<br>🇧🇷 Plataforma de microsserviços orientada a eventos (clientes, pedidos, faturamento) com **Spring Boot** e **Apache Kafka**. Os serviços se comunicam de forma assíncrona por tópicos Kafka; as notas fiscais são geradas com JasperReports e armazenadas no **MinIO**.
<br>`Java` · `Spring Boot` · `Kafka` · `PostgreSQL` · `MinIO` · `Docker`

#### [vitrine](https://github.com/JerryAdrianoo/vitrine)
🇺🇸 E-commerce marketplace backend in **pure Java**, emphasizing layered architecture and separation of concerns — multi-module design with JAX-RS/Jersey, service layer and a JPA/Hibernate persistence tier.
<br>🇧🇷 Backend de marketplace de e-commerce em **Java puro**, com foco em arquitetura em camadas e separação de responsabilidades — projeto multimódulo com JAX-RS/Jersey, camada de serviço e persistência com JPA/Hibernate.
<br>`Java` · `JAX-RS` · `Hibernate` · `Docker`

#### [order-suite](https://github.com/JerryAdrianoo/order-suite)
🇺🇸 Microservices architecture with **Java 21** and **Spring Boot 3+**: product, auth, inventory and notification services behind a **Spring Cloud Gateway**, orchestrated with Docker Compose.
<br>🇧🇷 Arquitetura de microsserviços com **Java 21** e **Spring Boot 3+**: serviços de produto, autenticação, estoque e notificações atrás de um **Spring Cloud Gateway**, orquestrados com Docker Compose.
<br>`Java 21` · `Spring Cloud Gateway` · `PostgreSQL` · `Docker`
